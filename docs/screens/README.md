# Screenshots

This directory is where redacted UI screenshots live for the README.
Placeholder — real assets should be added manually and every chat line
must be visually masked before commit.

Guideline:
- Only UI chrome (title bars, tab bars, buttons) may remain visible.
- Every message body / avatar / display name must be blanked or blurred.
- No timestamps that could correlate to a specific person.

Never commit raw runtime screenshots from `photos/` (excluded via
`.gitignore`).
